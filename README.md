# 一个前端开发学习<<数据结构与算法>>的心得与总结

## 前端开发为什么要学习<<数据结构与算法>>

很多人觉得数据结构与算法只有后端需要学习,前端不需要学习，那就大错特错了，因为现在好多业务逻辑都放在了前端，交互也会特别复杂，对数据的处理也比较多，所以我们很有必要去学习他
